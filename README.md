## edgeboard端网络连接优化
  首先就是大家都头疼的一个问题,网卡的适配问题.
  我们队先后尝试过官方的无线网卡,网线连接,其他驱动移植等方法,效果都不太理想.
  在经历了两天,拖着三米长的网线跟着车跑的尴尬局面之后,本人忍无可忍,想到了一个非常天才的解决方法,最终实现效果好极了.
